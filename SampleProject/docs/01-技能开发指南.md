# 技能开发指南

## 概述
技能（Skill）是 Agent 的核心能力模块，每个技能实现特定的功能领域。

## 技能模板

```python
# skills/my_skill.py
from typing import Dict, Any, List
from abc import ABC, abstractmethod

class BaseSkill(ABC):
    """技能基类"""
    
    def __init__(self, agent):
        self.agent = agent
        self.name = self.__class__.__name__
    
    @abstractmethod
    def execute(self, *args, **kwargs) -> Dict[str, Any]:
        """
        执行技能
        
        Returns:
            包含执行结果的字典:
            {
                "success": bool,
                "result": Any,
                "message": str,
                "data": Dict
            }
        """
        pass
    
    def get_description(self) -> str:
        """获取技能描述"""
        return f"技能: {self.name}"
    
    def get_examples(self) -> List[str]:
        """获取使用示例"""
        return []

class MySkill(BaseSkill):
    """自定义技能示例"""
    
    def execute(self, task: str, **kwargs) -> Dict[str, Any]:
        """
        执行技能
        
        Args:
            task: 任务描述
            **kwargs: 其他参数
            
        Returns:
            执行结果
        """
        try:
            # 实现具体逻辑
            result = self._process_task(task)
            
            return {
                "success": True,
                "result": result,
                "message": "执行成功",
                "data": {"task": task}
            }
        except Exception as e:
            return {
                "success": False,
                "result": None,
                "message": f"执行失败: {str(e)}",
                "data": {}
            }
    
    def _process_task(self, task: str) -> Any:
        """处理任务的具体实现"""
        # 你的逻辑代码
        return f"处理结果: {task}"
    
    def get_description(self) -> str:
        return "这是一个示例技能，用于演示如何开发新技能"
    
    def get_examples(self) -> List[str]:
        return [
            "skill.execute(task='分析数据')",
            "skill.execute(task='生成报告', format='pdf')"
        ]
```

## 注册技能到 Agent

```python
# 在 DeepSeekAgent 类中添加技能管理
class DeepSeekAgent:
    def __init__(self, api_key: str):
        # ... 现有代码 ...
        self.skills: Dict[str, BaseSkill] = {}
    
    def register_skill(self, skill: BaseSkill):
        """注册技能"""
        self.skills[skill.name] = skill
    
    def use_skill(self, skill_name: str, *args, **kwargs) -> Dict[str, Any]:
        """使用技能"""
        if skill_name in self.skills:
            return self.skills[skill_name].execute(*args, **kwargs)
        return {
            "success": False,
            "result": None,
            "message": f"技能 '{skill_name}' 未找到",
            "data": {}
        }
    
    def list_skills(self) -> List[str]:
        """列出所有技能"""
        return list(self.skills.keys())
```

## 使用示例

```python
from skills.my_skill import MySkill

# 创建 Agent
agent = DeepSeekAgent(api_key="your_api_key")

# 注册技能
my_skill = MySkill(agent)
agent.register_skill(my_skill)

# 使用技能
result = agent.use_skill("MySkill", task="执行某项任务")
if result["success"]:
    print(f"结果: {result['result']}")
else:
    print(f"错误: {result['message']}")
```

## 技能开发最佳实践

1. **单一职责**: 每个技能只做一件事
2. **错误处理**: 捕获异常并返回标准格式
3. **文档完善**: 添加清晰的 docstring
4. **可测试性**: 编写单元测试
5. **参数验证**: 验证输入参数
6. **日志记录**: 记录关键操作
