```
AgentDevelopment/
├── simple.py                    # 主程序入口
├── config.yaml                  # 配置文件（敏感信息，不提交）

├── README.md                    # 项目说明文档
│
├── skills/                      # 技能模块目录
│   ├── __init__.py
│   ├── reasoning.py            # 推理能力
│   ├── planning.py             # 规划能力
│   ├── memory.py               # 记忆管理
│   ├── learning.py             # 学习能力
│   └── communication.py        # 通信能力
│
├── tools/                       # 工具函数目录（用于 Function Calling）
│   ├── __init__.py
│   ├── web_tools.py            # 网络相关工具
│   ├── file_tools.py           # 文件操作工具
│   ├── data_tools.py           # 数据处理工具
│   ├── system_tools.py         # 系统相关工具
│   └── api_tools.py            # API 调用工具
│
├── docs/                        # 参考文档目录
│   ├── 00-项目结构说明.md       # 本文件
│   ├── 01-技能开发指南.md       # 如何开发新技能
│   ├── 02-工具开发指南.md       # 如何开发新工具
│   ├── 03-配置说明.md           # 配置文件详解
│   └── 04-最佳实践.md           # 开发最佳实践
│
├── examples/                    # 示例代码目录
│   ├── basic_usage.py          # 基础使用示例
│   ├── custom_skill.py         # 自定义技能示例
│   ├── custom_tool.py          # 自定义工具示例
│   └── advanced_agent.py       # 高级用法示例
│
├── tests/                       # 测试目录
│   ├── __init__.py
│   ├── test_skills.py          # 技能测试
│   ├── test_tools.py           # 工具测试
│   └── test_agent.py           # Agent 测试
│
├── logs/                        # 日志目录
│   ├── agent.log               # Agent 运行日志
│   └── error.log               # 错误日志
│
└── data/                        # 数据目录
    ├── memory/                 # 持久化记忆数据
    ├── cache/                  # 缓存数据
    └── exports/                # 导出数据
```

## 目录说明

### skills/ - 技能模块
存储 Agent 的各种核心能力，每个技能对应一个独立模块：
- **reasoning.py**: 逻辑推理、问题分析
- **planning.py**: 任务规划、方案制定
- **memory.py**: 长期记忆、知识管理
- **learning.py**: 从经验中学习
- **communication.py**: 多模态交互能力

### tools/ - 工具函数
Function Calling 所用的函数集合，按功能分类：
- **web_tools.py**: HTTP 请求、爬虫、搜索
- **file_tools.py**: 文件读写、格式转换
- **data_tools.py**: 数据处理、分析、可视化
- **system_tools.py**: 系统命令、环境信息
- **api_tools.py**: 第三方 API 调用封装

### docs/ - 参考文档
项目相关的文档和开发指南：
- 技能开发指南
- 工具开发指南
- 配置说明
- 最佳实践

### examples/ - 示例代码
展示如何使用 Agent 框架的示例：
- 基础用法
- 自定义技能
- 自定义工具
- 高级特性

### tests/ - 测试代码
单元测试和集成测试：
- 技能测试
- 工具测试
- Agent 整体测试

### logs/ - 日志目录
运行日志和错误日志：
- agent.log: 常规日志
- error.log: 错误日志

### data/ - 数据目录
Agent 的数据存储：
- memory/: 持久化记忆
- cache/: 临时缓存
- exports/: 导出数据
